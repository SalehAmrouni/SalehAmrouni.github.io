
# Student Atlas TSA Webmaster 2026

## Project Description
Student Atlas is a web application designed for the TSA Webmaster 2026 competition. It provides a modern, accessible, and research-driven platform for student opportunity management, featuring a clean UI, accessibility options, and AI-powered features.

## Technical Stack
- HTML5 Semantic Markup
- CSS3 with Custom Properties
- Vanilla JavaScript (ES6+)
- Lucide Icons (ISC License)
- Compromise NLP (MIT License)
- OpenDyslexic Font (Accessibility)
- Custom AI Engine (Zeus AI)

## Features
- Modern, responsive UI
- Accessibility customization (font size, color, dyslexia-friendly, etc.)
- AI-powered suggestions and search
- Profile and opportunity management
- PDF documentation and work log

## Usage
Open index.html in a browser or deploy to a static web server.
